# Wi-Fi QR Scanner & Password Recovery App

A modern, web-based **Wi-Fi QR scanner** that lets you recover Wi-Fi details like **SSID, password, encryption type, hidden status, and raw QR data**. The app supports scanning **QR codes from images** or **live camera scanning**. It also keeps a **scan history** and can **export it as JSON or CSV**.  

This project is built with **HTML, CSS, and JavaScript**, and uses [jsQR](https://github.com/cozmo/jsQR) for QR code decoding. Itâ€™s also a **PWA-ready web app**, so you can install it on your device like a native app.

---

## Features

- Scan Wi-Fi QR codes from **image files** or **live camera feed**.  
- Display full Wi-Fi details:
  - SSID  
  - Password  
  - Encryption type  
  - Hidden network  
  - Raw QR data  
  - Scanned timestamp  
- Keep a **scan history** with export options (JSON / CSV).  
- Clear scan history with one click.  
- **Dark / Light theme** toggle.  
- Installable as a **Progressive Web App (PWA)**.  
- Social media links in the footer using **vector icons**.
